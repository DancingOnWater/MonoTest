
Инструкция по сборке и запуску испытательного ПО Mono
0. Скопировать Npgsql.dll, Entityframework.dll, Entityframework6.Npgsql.dlll в дирректорию с проектом
1. Установить сервер БД postgresql.
1.1 Восставноить БД из файла TestBaseBackup
2. В файле {путь к проекту}/testMono/Config.cs в строке:

	"Host={адрес сервера};Database={имя БД};Username={имя пользователя};Password={пароль};",

	заменить "{адрес сервера}", "{имя БД}", "{имя пользователя}", "{пароль}" на
	адрес сервера Postgresql, имя обслуживаемой БД, имя пользователя БД, пароль пользователя БД.

3. Установить сервер БД mssql
4. В файле {путь к проекту}/testMono/Program.cs в строке:

	@"Data Source = {адрес сервера}\{экземпляр сервера}; Initial Catalog = {имя БД}; Persist Security Info = True; User ID = {имя пользователя}; PWD = {пароль};",

	заменить "{адрес сервера}", "{экземпляр сервера}", "{имя БД}", "{имя пользователя}", "{пароль}" на 
	адрес и имя экземпляра сервера mssql, имя обслуживаемой БД, имя пользователя БД, пароль пользователя БД.

5. Далее выполнить следующие действия:
5.1. Открыть консоль и перейти в директорию с проектом
5.2. Выполнить команду msbuild -t:restore
5.3. Выполнить команду msbuild testMono.sln -t:Rebuild -p:Configuration=Release
5.4. Выполнить команду mono ./testMono/bin/Release/net472/testMono.exe
 
При появлении ошибок, в консоль будет выведены красным цветом сообщения об ошибках.
